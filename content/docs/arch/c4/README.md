## Structurizr Lite Webserver

```bash
# View at: http://localhost:8080/workspace/diagrams#SystemContext

# DSE rendering.
$ make lite
# FSIL redering.
$ WORKSPACE=fsil_workspace make lite

# Stop the web server
$ make stop
```
